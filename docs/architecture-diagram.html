HOW IT WORKS

  • The engine is fully deterministic — no LLM, no API keys, no rate limits, so audits never stall.
  • Each page is crawled, then 60 test cases run per page and at 6 screen resolutions (640×480 → 1920×1080, tablet, mobile).
  • Every finding is attributed to its own page and carries that page’s evidence screenshot.
  • Results stream live to the UI and persist in the browser; the full report exports to Excel / CSV.
